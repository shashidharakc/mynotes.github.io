Back Prop

Lets see how we can update/adjust the weights based on the information we have in hand. So we update given weight, lets say WjW_{j}Wj​, by deriving partial derivative of loss function w.r.t that weight WjW_{j}Wj​, i.e ∂Etotal∂Wj\frac{\partial E_{total}}{\partial W_{j}}∂Wj​∂Etotal​​. Using chain rule we can get partial derivative loss EtotalE_{total}Etotal​ w.r.t WjW_{j}Wj​ as shown below.

∂Etotal∂Wj=∂Etotal∂yHat∗∂yHat∂Zi1∗∂Zi1∂Wj\frac{\partial E_{total}}{\partial W_{j}} = \frac{\partial E_{total}}{\partial yHat} * \frac{\partial yHat}{\partial Z^{1}_{i}} * \frac{\partial Z^{1}_{i}}{\partial W_{j}}∂Wj​∂Etotal​​=∂yHat∂Etotal​​∗∂Zi1​∂yHat​∗∂Wj​∂Zi1​​

Based on the derivative of Cross-entropy with softmax, we can replace the first two terms of the right hand side as.

∂Etotal∂yHat∗∂yHat∂Zi1=(y^i−yi)\frac{\partial E_{total}}{\partial yHat} * \frac{\partial yHat}{\partial Z^{1}_{i}} = (\hat y_{i} - y_{i})∂yHat∂Etotal​​∗∂Zi1​∂yHat​=(y^​i​−yi​)

Substituting the same we get,

∂Etotal∂yHat∗∂yHat∂Zi1∗∂Zi1∂W1=(y^i−yi)∗∂Zi1∂W1\frac{\partial E_{total}}{\partial yHat} * \frac{\partial yHat}{\partial Z^{1}_{i}} * \frac{\partial Z^{1}_{i}}{\partial W_{1}} = (\hat y_{i} - y_{i}) * \frac{\partial Z^{1}_{i}}{\partial W_{1}}∂yHat∂Etotal​​∗∂Zi1​∂yHat​∗∂W1​∂Zi1​​=(y^​i​−yi​)∗∂W1​∂Zi1​​

∂Zi1∂Wj=∂W1ij∗X1+∂W2ij∗X2+∂W3ij∗X3+∂W4ij∗X4∂Wj\frac{\partial Z^{1}_{i}}{\partial W_{j}} = \frac {\partial W_{1}^{ij} * X_{1} + {\partial W_{2}^{ij}} * X_{2} + {\partial W_{3}^{ij}} * X_{3} + {\partial W_{4}^{ij}} * X_{4}} {\partial W_{j}}∂Wj​∂Zi1​​=∂Wj​∂W1ij​∗X1​+∂W2ij​∗X2​+∂W3ij​∗X3​+∂W4ij​∗X4​​
